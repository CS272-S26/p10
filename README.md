For this project I will be creating a mock e-commerce website that is selling some products. 

I will be using RainForest API to get product listings, which will also allow users to search for any products they desire (as long as its available on Amazon since RF API scrapes amazon)

The project will have 7 pages -

index.html - This will serve as the homepage/landing page of the website with links to the contact us section and about us section along with a preview of some featured products

cart.html - This is will the cart that will show all the saved items along with the quantity and price total for all of them, this will work by using the local storage data that is saved by the user earlier, otherwise the cart will be empty

about.html - This will be the section that tells the user about us, our mission and why this website exists

products.html - This is our search section where the user can search for the items they specifically need

product.html - This is the description page for an individual product, it updates and changes for each product to display the individual unique details they have along with a few top reviews 

contact-us.html - This page has a simple form to contact us through the website itself along with some information to allow them to contact us throughother modes like calling, email, etc

checkout.html - This is the checkout page that allows the user to see the whole checkout summary, including the items and the the total
